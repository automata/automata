Genetic programming to contours generation

We were working on a genetic programming algorithm to generate contours based on a given example.

Each line is a population ordered from best to poor fitness. Each individual have a genotype (draw instructions) and a phenotype (the drawing itself).

It is interesting to see how individuals make clusters of similar contours evolving to a ‘zig-zag’ pattern.

Some contours resemble 1950's works byManfred Mohr.

@Flowhub_io, @thegrid and chimarrão. Nice match for a rainy day.
